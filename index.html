Posting Data

Once we have created an account for you, you will get a scorecards ID, an appname (name of your application within Call Criteria) and an API key. To POST data to us, you would add the appname and API key to the URL in this format:

http://app.callcriteria.com/callcriteriaAPI.svc/json/AddRecord?
appname=<appname>&apikey=<apikey>

or

http://app.callcriteria.com/callcriteriaAPI.svc/xml/AddRecord?
appname=<appname>&apikey=<apikey>

Post for data in either json format or XML format. Simple change the posting URL to use json or xml as above.

Usage

The following fields are available, but not required, to post data to. If the data you are posting does not have the same fields, but you would like to use an existing field – feel free to map to that field. If it is necessary to add fields, please contact us at Stace@CallCriteria.com with any concerns or comments.

Online docs are available with prototypes and samples of XML and JSON posts at https://app.callcriteria.com/callcriteriaAPI.svc/json/help
